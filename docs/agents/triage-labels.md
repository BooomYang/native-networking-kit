# Triage labels

相关 engineering skills 使用五个 canonical triage roles。本 repo 先采用默认 label vocabulary；如果 GitHub repo 中实际 label 名称不同，后续只改本文件右列即可。

| Skill role | GitHub label | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要 maintainer 评估 |
| `needs-info` | `needs-info` | 等待 reporter 补充信息 |
| `ready-for-agent` | `ready-for-agent` | 需求已足够明确，可交给 agent 执行 |
| `ready-for-human` | `ready-for-human` | 需要 human implementation 或判断 |
| `wontfix` | `wontfix` | 明确不处理 |

当 skill 提到某个 role，例如 “apply the ready-for-agent triage label”，使用本表对应的 GitHub label。
